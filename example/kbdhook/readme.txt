Keyboard hook example for Borland C++Builder3.
Written by Michael Tsai (easyman@ms2.seeder.net) Feb-16-1999.

1. Open MakeAll.bpg in the IDE. Build KbdHook.bpr first, then TestHook.bpr.
2. Run TestHook.exe, press 'Install hook' button, now execute NotePad.exe or
   WORD.EXE, press Ctrl+Alt+A then you can see what is this example for.
